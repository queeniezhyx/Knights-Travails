# Knights Travails

## Project Goal
A logic-based project focused on **Graph Traversal** and **Breadth-First Search (BFS)** algorithms. The goal was to build a function that identifies the shortest path for a Chess Knight to travel between any two squares on a standard 8x8 board.

This project covers the implementation of:
- **Graph Representation:** Treating a chessboard as an implicit graph where squares are nodes and legal moves are edges.
- **BFS Algorithm:** Implementing a search that guarantees the shortest path in an unweighted graph by exploring moves layer by layer.
- **Coordinate Logic:** Handling 2D array transformations and validating moves within the legal 0-7 board boundaries.
- **Visited Tracking:** Managing a stateful record of visited coordinates to prevent infinite loops and redundant calculations.
- **Path Reconstruction:** Storing and returning the full sequence of moves from the starting square to the target destination.

## Tech Stack
- **JavaScript** (ES6 Modules)
- **Node.js** (Runtime Environment)
